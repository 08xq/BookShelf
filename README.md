# 书柜
一款用于管理个人图书及阅读笔记的移动 Android 应用。

图书查询使用了 Volley 网络库调用豆瓣图书API。

# 运行环境
Android 4.3 及以上版本

# 功能特色
- 支持通过扫一扫 ISBN 书号快速录入图书 - 支持通过在线搜索书名录入图书
- 支持图书收藏功能
- 支持图书笔记功能
- 支持图书分享功能
